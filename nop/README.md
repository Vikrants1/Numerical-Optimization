# Car Price Prediction Project

This project predicts the resale value of used cars using machine learning.

## Main Features
- CLI-based user input for custom prediction
- Comparison of LASSO Regression and Linear Regression
- Data preprocessing with categorical encoding
- Visualization of actual vs predicted prices

## Expected Dataset Columns
- Car_Name
- Year
- Selling_Price
- Present_Price
- Kms_Driven
- Fuel_Type
- Seller_Type
- Transmission
- Owner

## How to Run
1. Install dependencies from requirements.txt
2. Run the main Python file
3. Enter car details in the terminal
4. View the model comparison and predicted resale price
